# Electric Plan Specifications

## General Notes
1. All electrical work must comply with the current National Electrical Code (NEC) and local building codes.
2. Verify all device locations with owner/architect prior to rough-in.

## Wiring Requirements

### Ceiling Fan Control
To ensure independent control of fan and light functions (or for future smart control integration), an extra conductor is required for all ceiling fan switch legs.
- **Single Pole Locations:** Run **14-3** (or 12-3 as required by circuit).
- **3-Way Locations:** Run **14-4** (or 12-4 as required by circuit).

### Low Voltage / Data
- **Cat-6 Locations:** All data drops shall be run in **1" flexible conduit** from the device box to the basement utility area.

## Circuiting
1. Kitchen small appliance circuits shall be 20A.
2. Bathroom receptacles shall be on dedicated 20A GFCI protected circuits.
3. Smoke and Carbon Monoxide detectors shall be hardwired with battery backup and interconnected.
