# Plumbing and HVAC

## HVAC Requirements
- 1 zone per floor.
- If HVAC contractor finds it suitable, provide a unit per floor.
- **Proposed Unit Sizing:**
    - First Floor: 2.5 Ton
    - Second Floor: 1.5 Ton
    - *Note: Sizing is preliminary and must be confirmed by the HVAC Contractor.*

## Plumbing Requirements
- **System Type:** Hybrid home run system.
- **Manifold:** Central manifold (Contractor option: Site-built or Pre-Manufactured).
- **Trunk Lines (Manifold to Room):**
    - Hot: 1/2" run to each room/group.
    - Cold: 3/4" run to each room/group.
- **Fixture Branch Lines (In-room to Fixture):**
    - Hot: 3/8" supply to each fixture.
    - Cold: 1/2" supply to each fixture.
